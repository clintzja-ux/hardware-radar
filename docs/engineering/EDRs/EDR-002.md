EDR-002

Title

Decision Aggregation Is Pure

Decision

DecisionAggregator is a pure function over a completed ValidationRun.

It:

has no side effects,
performs no I/O,
mutates nothing,
always returns the same output for the same input.

Reason

Pure functions are easy to:

test,
reason about,
cache,
audit,
replay,
verify.

This is exactly the kind of component that benefits from functional behavior, especially in a validation pipeline where determinism and auditability are critical.
