# Executive Summary

Hardware Radar is a live public website focused on helping users find the cheapest verified RAM prices from trusted retailers.

The public website is currently available at:

**https://cheapestram.com**

The current release uses placeholder pricing and demonstration data while the underlying platform architecture is being developed.

Development is now focused on transforming the existing website into a scalable hardware intelligence platform capable of supporting live pricing, recommendations, affiliate integrations, and future hardware categories.
