Chapter 4 — Writing Style

These are some of my favorite rules we've developed.

HR-W001

Always show real dollar amounts.

Never emphasize percentage savings.

Example:

✔ Save $18

Instead of:

✘ Save 23%

HR-W002

Use plain English.

Instead of:

CAS latency optimization

Say:

Lower latency for gaming.

HR-W003

Avoid marketing language.

Never write:

Revolutionary!

Instead:

Best value today.

HR-W004

Write like a trusted friend.

Not like a salesperson.
