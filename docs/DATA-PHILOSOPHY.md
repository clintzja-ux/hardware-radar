Hardware Radar Data Philosophy
Facts over opinions.

Manufacturer specifications are facts.

Observations are facts.

Recommendations are derived from facts.

Opinions are never stored.

Every fact has evidence.

If Hardware Radar cannot identify where a fact came from,

it should not exist.

One product.

One identity.

Forever.

Products never change identity.

Observations change constantly.

Explain every recommendation.

Every recommendation must be traceable back to Atlas facts and Mercury observations.

The website is not the product.

The catalog is the product.

The website is merely one way to access it.
