# Hardware Radar Utilities

This directory contains small, reusable helper functions shared across the Hardware Radar frontend.

Utilities should:

- Be pure functions whenever possible.
- Have no knowledge of the DOM.
- Have no knowledge of Atlas, Mercury, or Compass.
- Perform one clearly defined task.
- Be safe to reuse anywhere in the application.

Examples:

- Text formatting
- Date formatting
- Currency formatting
- Unit formatting
- Validation
